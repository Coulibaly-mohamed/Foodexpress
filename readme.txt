# 🍔 FoodExpress API

Une API RESTful pour gérer des restaurants et leurs menus. Authentification via JWT. Documentation Swagger intégrée.

## 📦 Stack technique

- Node.js + Express.js
- MongoDB + Mongoose
- JSON Web Tokens (JWT)
- Swagger (OpenAPI) pour la documentation
- Dotenv pour la configuration

---

## 🚀 Démarrage

### Prérequis

- Node.js v18+
- MongoDB local ou en ligne
- `npm install` pour installer les dépendances

### Installation

```bash
git clone https://github.com/votre-utilisateur/foodexpress-api.git
cd foodexpress-api
npm install
